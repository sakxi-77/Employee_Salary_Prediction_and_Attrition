# Employee Salary & Attrition Prediction System

## Overview

A Machine Learning project that predicts employee salary levels and employee attrition using HR analytics data. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and a Flask-based web application for predictions.

## Features

- Employee Attrition Prediction
- Salary Prediction
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Interactive Prediction Interface

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Flask
- Jupyter Notebook

## Machine Learning Workflow

- Data Cleaning
- Missing Value Handling
- Label Encoding
- Feature Selection
- Model Training
- Model Evaluation
- Model Deployment
